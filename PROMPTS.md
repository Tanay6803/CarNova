## Project Name

CarNova

---

## Project Description

CarNova is a modern Vehicle Inventory and Dealership Management System developed as an MCA Major Project.

The system provides secure authentication, inventory management, vehicle management, stock monitoring, image uploads, dashboard analytics, and purchase simulation.
